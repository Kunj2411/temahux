# Temahux User Journey Map

## 🎯 Three Primary User Personas

### 1. Business Owner (Services Path)
**Goal**: Grow business online
**Pain Points**: Need website, social media, marketing help
**Journey**: Homepage → Services → Package Selection → Consultation

### 2. Institution Decision Maker (Products Path)
**Goal**: Modernize institutional systems
**Pain Points**: Outdated software, manual processes
**Journey**: Homepage → Products → Early Access Signup

### 3. Career Seeker (Academy Path)
**Goal**: Learn future tech skills
**Pain Points**: Need practical skills, career change
**Journey**: Homepage → Academy → Waitlist Signup

## 🛤️ Path 1: Business Owner Journey

```
HOMEPAGE
   │
   ├─ Sees "Grow My Business Online" card
   │
   ▼
SERVICES PAGE (/services)
   │
   ├─ Reads hero: "Growth Infrastructure"
   ├─ Scrolls to services grid
   │
   ▼
SERVICE SELECTION
   │
   ├─ Hovers over cards to see features
   ├─ Sees pricing (₹3,000 - ₹15,000+)
   │
   ▼
PACKAGE COMPARISON
   │
   ├─ Compares Starter vs Business vs E-Commerce
   ├─ Sees "Popular" badge on Business package
   │
   ▼
PORTFOLIO REVIEW (Optional)
   │
   ├─ Views recent work examples
   ├─ Builds trust
   │
   ▼
CONVERSION POINT
   │
   ├─ Clicks "Start My Project" OR
   ├─ Clicks "Free Consultation" OR
   ├─ Clicks Floating CTA button
   │
   ▼
CONSULTATION FORM
   │
   ├─ Fills name, email, company
   ├─ Selects service interest
   ├─ Provides budget range
   │
   ▼
SUBMISSION
   └─ Lead captured ✅
```

### Key Decision Points:
1. **Service Selection**: Which service do I need?
2. **Budget Fit**: Can I afford this?
3. **Trust Building**: Have they done this before?
4. **Action**: Ready to start?

### Conversion Triggers:
- Clear pricing (no surprises)
- Feature transparency (hover reveals)
- Social proof (portfolio)
- Multiple CTAs (low friction)
- Free consultation (low commitment)

## 🔄 Re-engagement Strategies

### Bounce Prevention
1. **Immediate Clarity**: "What brings you here?" section
2. **Visual Hierarchy**: Large, clear cards
3. **Multiple Entry Points**: Pillars + Cards + Nav

### Exit Intent (Future Enhancement)
- Modal with special offer
- "Before you go" message
- Newsletter signup

### Return Visitors
- Remember last visited section
- Show "New" badges on updates
- Personalized recommendations

## 🎨 Emotional Journey

### Homepage (Discovery)
**Emotion**: Curiosity, Exploration
**Design**: Bold, expansive, three clear paths
**Goal**: Help user self-identify

### Service/Product/Academy Pages (Evaluation)
**Emotion**: Consideration, Comparison
**Design**: Detailed, transparent, trustworthy
**Goal**: Build confidence in solution

### Conversion Points (Decision)
**Emotion**: Commitment, Excitement
**Design**: Clear CTAs, low friction, reassuring
**Goal**: Make action easy

### Post-Conversion (Confirmation)
**Emotion**: Satisfaction, Anticipation
**Design**: Thank you message, next steps
**Goal**: Set expectations
